# Installation Guide

## Setup Instructions

```bash
composer install
```

```bash
php artisan migrate
```

```bash
php artisan db:seed --class=UserSeeder
```

```bash
npm install
```

```bash
php artisan serve
```

```bash
npm run dev
```
